OpenMeteoData

Um projecto aberto que visa tornar os dados meteorológicos acessíveis a todos.

Após um ano de experiência bem sucedida com pilotos franceses de parapentes e de planadores, actualmente este projecto encontra-se robusto, pelo que se estende agora a toda a Europa.

Disponibilizamos:

  • Um motor de pesquisa que permite procurar e transferir, de forma fácil, os dados abertos das instituições científicas.
  • O nosso próprio super-computador para fazer previsões meteorológicas de forma aberta.
  • Uma força política que luta pelo acesso a dados de serviços meteorológicos públicos (ECMWF, Météo France, MetOffice...).
  • Uma plataforma para ajudar as instituições a disponibilizar os seus dados (open data).
  • Uma organização sem fins lucrativos que visa agregar outros projectos com dados abertos na área da meteorologia.
